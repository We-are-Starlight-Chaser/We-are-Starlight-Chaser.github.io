Simple Minecraft Server Manager

SMSM 是一个面向 Windows 系统的开源 Minecraft 本地服务器管理器， 由 C# + WPF 驱动，为本地服务器打造一站式开服解决方案。 支持多服务器管理、JDK 智能管理、可视化控制台、自动备份等功能。
